__attribute__ ( (section(".os"))  )

A generic thread create api.

Resume a previously suspended thread by its ID.

Suspend an arbitrary thread by its ID.

Suspend the calling thread indefinitely.

Block the calling thread for ms milliseconds.

Delete a thread by ID and free its resources.

Note
Creates a dynamic FreeRTOS task and inserts the new thread_handle_t into the intrusive doubly-circular thread_list using list_add_tail(), mirroring the Linux kernel task list.
Parameters
thread_funcThread entry function pointer.
thread_nameName string for debugging.
thread_stack_depthStack size in words.
priorityScheduling priority (0 = idle).
thread_parameterOpaque pointer forwarded to the thread.
Returns
Positive thread_id on success, negative OS_ERR_* on failure.
Note
Walks the intrusive list with list_for_each_entry(), calls vTaskDelete(), unlinks via list_delete(), then frees.
Parameters
thread_idID returned by os_thread_create().
Returns
OS_ERR_NONE on success, OS_ERR_OP if the ID is not found.
Note
Equivalent to vTaskSuspend(NULL) — the thread must be resumed externally via os_resume_thread().
Walks the intrusive thread list to locate the handle then calls vTaskSuspend(), mirroring how the Linux kernel iterates task_struct via the tasks list.
Parameters
thread_idID returned by os_thread_create().
Note
Walks the intrusive thread list and calls vTaskResume().
Parameters
thread_idID returned by os_thread_create().

struct list_node thread_list = { &( thread_list ), &( thread_list ) }

This section contains all the static varibles used in this file.

variables declaration

Sentinel list node for the doubly-circular thread list.

Note
Follows the Linux kernel intrusive list pattern: thread_handle_t embeds a struct list_node so no separate pointer bookkeeping is needed. An empty list has thread_list.next == thread_list.prev == &thread_list.
