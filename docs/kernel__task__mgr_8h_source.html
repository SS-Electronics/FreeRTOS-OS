1/*
2 * kernel_task_mgr.h — Public API for the kernel task health monitor
3 *
4 * This file is part of FreeRTOS-OS Project.
5 *
6 * Exposes the snapshot types and accessor used by
7 * services/kernel_service_task_manager.c. Consumers (e.g. the shell)
8 * call task_mgr_get_health() to read the last periodic scan result.
9 *
10 * Thread safety note: the snapshot is refreshed every TASK_MGR_SCAN_PERIOD_MS
11 * by the task-manager thread and is read without a lock by the shell. On a
12 * single-core Cortex-M4 under FreeRTOS this may produce a slightly torn read
13 * if a scan occurs concurrently; for a debug/diagnostic interface this is
14 * acceptable.
15 */
16
