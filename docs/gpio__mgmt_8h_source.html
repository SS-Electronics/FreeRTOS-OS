1/*
2 * gpio_mgmt.h — GPIO management service thread
3 *
4 * This file is part of FreeRTOS-OS Project.
5 *
6 * The GPIO management thread owns logical GPIO line handles and exposes a
7 * simple message-based API for setting, clearing, toggling, and reading
8 * GPIO pins from any FreeRTOS task context.
9 *
10 * Unlike UART/SPI/I2C, GPIO operations are very fast, so most callers will
11 * call the generic gpio driver directly (drv_gpio_set / drv_gpio_clear).
12 * The management thread is useful for:
13 * • Delayed GPIO operations (e.g. "assert reset for 10 ms, then release").
14 * • Scheduled blinking / heartbeat LEDs.
15 * • Centralised GPIO registration at startup.
16 */
17
