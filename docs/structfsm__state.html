Detailed Description

A single state in the FSM / HSM.

Set 'parent' to enable hierarchical (nested) states — the FSM will automatically call exit/entry handlers in the correct top-down order.

Static initialiser example:

static fsm_state_t st_idle = {
.id = ST_IDLE,
.parent = NULL,
.on_entry = idle_entry,
.on_exit = NULL,
};
