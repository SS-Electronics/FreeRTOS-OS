◆ FSM_HIERARCHY_DEPTH_MAX

#define FSM_HIERARCHY_DEPTH_MAX   8U

Maximum number of ancestor states (HSM nesting depth).

Increase if you have deeply nested hierarchical states. Each level costs one pointer slot on the stack during transitions.

Default: 8

◆ FSM_THREAD_SAFE

#define FSM_THREAD_SAFE   0

Enable mutex-based protection of the internal event queue.

Set to 1 when the FSM is accessed from:

  • Multiple RTOS tasks
  • A task AND an ISR simultaneously

Set to 0 for bare-metal single-context use (no overhead).

When enabled, provide lock/unlock hooks via fsm_set_lock() after fsm_init().

Default: 0
