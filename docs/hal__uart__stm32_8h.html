const drv_uart_hal_ops_t * hal_uart_stm32_get_ops ( void  )

Return a pointer to the STM32 UART HAL ops table.

Pass this to drv_uart_register() to bind STM32 HAL functions to a generic UART handle. The returned pointer is to a static const struct — do not free or modify it.
