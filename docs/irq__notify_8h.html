typedef void(* irq_notify_cb_t) (irq_id_t id, void *data, void *arg, BaseType_t *pxHPT)

irq_notify_cb_t — subscriber callback invoked by irq_notify / irq_notify_from_isr.

Parameters
idThe IRQ ID that fired.
dataEvent-specific payload pointer (e.g. received byte ptr, error code ptr).
argUser-supplied argument passed to irq_register().
pxHPTFreeRTOS higher-priority-task-woken accumulator. Set via vTaskNotifyGiveFromISR() etc. if a task was unblocked. NULL when called from task context (irq_notify).
