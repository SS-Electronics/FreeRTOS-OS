int32_t os_shell_mgmt_start ( void  )

Create and start the OS shell management task.

Must be called once from the OS init sequence before the scheduler starts. The task will delay TIME_OFFSET_OS_SHELL_MGMT ms before printing the banner and entering the read loop, ensuring the UART driver is initialised first.

Returns
OS_ERR_NONE on success, OS_ERR_OP if task creation fails.

static BaseType_t os_shell_register_command ( const CLI_Command_Definition_t *  cmd)
inlinestatic

Register a custom FreeRTOS+CLI command with the shell.

Thin wrapper around FreeRTOS_CLIRegisterCommand() so callers only need to include this header. Must be called before os_shell_mgmt_start(), or at any time from a task (FreeRTOS+CLI uses a linked list protected at the application level).

Parameters
cmdPointer to a const CLI_Command_Definition_t in static storage.
Returns
pdTRUE on success, pdFALSE if registration fails.
