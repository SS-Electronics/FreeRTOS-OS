int32_t uart_mgmt_start ( void  )

Create and start the UART management thread.

Call this once from the OS init sequence, before the FreeRTOS scheduler starts. The thread will perform hardware initialisation after the scheduler starts (after the startup delay defined in os_config.h).

Returns
OS_ERR_NONE on success, negative OS_ERR_* on failure.
