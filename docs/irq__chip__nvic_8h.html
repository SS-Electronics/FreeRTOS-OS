Bind logical IRQ to NVIC hardware interrupt.

irq_chip_nvic_bind_hwirq — associate a software IRQ ID with an NVIC line.

Sets irq_data.hwirq to irqn and configures priority. Must be called before irq_enable() so the chip ops have a valid hwirq to act on.

Parameters
irqSoftware IRQ ID.
irqnHardware NVIC IRQn number (e.g. USART1_IRQn).
priorityNVIC preempt priority.
irqLogical IRQ ID
irqnNVIC IRQ number
priorityInterrupt priority

Associates a generic IRQ descriptor with a physical NVIC line and initializes its priority.

Precondition
irq must map to a valid irq_desc
