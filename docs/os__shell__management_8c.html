◆ os_shell_mgmt_start()

int32_t os_shell_mgmt_start ( void  )

Create and start the OS shell management task.

Must be called once from the OS init sequence before the scheduler starts. The task will delay TIME_OFFSET_OS_SHELL_MGMT ms before printing the banner and entering the read loop, ensuring the UART driver is initialised first.

Returns
OS_ERR_NONE on success, OS_ERR_OP if task creation fails.
